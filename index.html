Child's name application:

Child's name application is the app that can show you the name you can give the child accordingly to the day he/she was born and his/her gender and it is application is for Ghanaian people. The form below contains two input elements; the first element is where you will fill your Akan's birthday date, month and the year and the second element is where you will select the gender of your Akan so that you can see the name that you can give to him/her:
